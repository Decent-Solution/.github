# Support

Thanks for using [@Decent-Solution](https://github.com/Decent-Solution) projects. Here is how to get help.

## 📖 Documentation

Start with the repository's `README` and any docs in the project. Many questions are answered there.

## 🐛 Bugs & feature requests

- Search [existing issues](https://github.com/Decent-Solution) first — your question may already be answered.
- If not, open a new issue using the appropriate template (bug report or feature request).

## 💬 General questions

For questions that are not bugs or feature requests, please use the repository's **Discussions** tab where enabled.

## 🏢 Commercial & enterprise support

Decent Solution offers consulting, AI, cloud, and managed services. For commercial enquiries, partnerships, or enterprise support:

- 🌐 **Website:** [decentsolution.cloud](https://decentsolution.cloud)
- ✉️ **Email:** developers@decentsolution.cloud

## 🔒 Security issues

Please **do not** use public issues for security vulnerabilities. Follow our [Security Policy](SECURITY.md).
